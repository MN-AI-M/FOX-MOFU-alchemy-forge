About This Tool / このツールについて

「Fox Mofu Pixel Art Converter」は、ブラウザゲーム Bloxd.io の建築向けに特化した、高精度の画像変換（Image Alchemy）ツールです。CORE FORGEエンジンを搭載し、お手持ちの画像を瞬時にプロ級のピクセルアート設計図に変換します。

Features / 主な機能

  • 340 Blocks Supported: Bloxd.io内に存在する340種類ものブロックデータを網羅。色合いや質感を極限まで再現します。
  • High-Performance Algorithm: 独自の計算により、最も自然で美しいブロックの組み合わせを自動提案します。
  • Easy to Use: 画像をアップロードするだけで、誰でも簡単に高品質なドット絵（Pixel Art）の設計図を作成可能です。

How to Use / 使い方

上のツール画面から変換したい画像（png, jpg等）をアップロードしてください。設定を調整することで、出力されるブロックの解像度や使用するブロックの種類をコントロールできます。
